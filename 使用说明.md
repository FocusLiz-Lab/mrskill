# mrskill 使用说明

## 这是什么

`mrskill` 是 Mel Robbins 播客访谈与自我改变主题的 Agent Skill 工具箱。它默认使用 IMA 知识库：

```text
Mel Robbins 知识库 | 播客访谈
```

你可以用它学习播客内容、把观点转成行动练习、做个人卡点复盘，或把播客主题改写成内容选题和脚本。

## 适合谁

- 想系统学习 Mel Robbins 播客、书籍和访谈主题的人
- 想把行动力、自信、拖延、边界、关系和情绪管理方法变成日常练习的人
- 想基于播客资料做内容选题、脚本、摘要或学习笔记的人
- 想让 Agent 帮忙从 IMA 知识库里检索相关资料的人

## 安装方式

发布后执行：

```bash
npx -y skills add FocusLiz-Lab/mrskill -g --all
```

安装完成后，直接用 `$mrs` 或具体子 skill 调用。

## IMA 知识库配置

使用前需要：

- 已安装官方 `ima-skill`
- 已配置 IMA `Client ID` 和 `API Key`
- 当前 IMA 账号有权限访问 `Mel Robbins 知识库 | 播客访谈`

如果没有安装：

```text
请安装 ima 技能
下载地址：https://app-dl.ima.qq.com/skills/ima-skills-1.1.7.zip
API Key 获取：https://ima.qq.com/agent-interface
```

## 典型场景

### 学习路径

先用 `$mrs-learning-map` 建一个主题地图，再选 1 个主题进入 `$mrs-practice` 做行动练习。

### 个人复盘

当你描述一个真实卡点时，用 `$mrs-coach`。它会先定位模式，再给下一步动作。

### 内容创作

如果要做短视频、newsletter 或播客笔记，用 `$mrs-content`。它会优先提炼场景、冲突、洞察和行动提示，不搬运原文。

### 查资料

如果你需要具体来源、节目、嘉宾、原始观点或出处，用 `$mrs-ima` 先检索。

## 常见问题

### 每次都要输入知识库名称吗？

不用。所有 workflow 默认使用：

```text
Mel Robbins 知识库 | 播客访谈
```

### 可以直接引用播客原文吗？

不建议直接长引用。需要引用时必须先通过 IMA 检索确认，并保持短引用。一般输出应以摘要、转述、评论和行动建议为主。

### 这是心理咨询吗？

不是。`mrs-coach` 只能做反思、提问和行动建议，不能替代心理治疗、医学诊断或危机干预。

## 发布边界

本仓库只发布 skill instructions、workflow 抽象和种子原子库，不发布原书、PDF、播客转录稿、YouTube 文稿、X/Twitter 归档、IMA 私有导出或任何未授权资料。
